## üèÉ Getting Started

`Required Node.js version 21.4 and higher!`

To get a local copy of the project perform the following actions:

1. Clone the repo

```
git clone https://github.com/iGafar/scan-react.git
```

2. Install npm packages

```
npm install
```

3. Run the project locally

```
npm run dev
```
